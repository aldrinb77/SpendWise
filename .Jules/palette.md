## 2026-04-18 - Dynamic ARIA Labeling\n**Learning:** When adding ARIA labels to buttons containing dynamic visual text (like filter dropdowns), replacing the content entirely with a static ARIA label is an accessibility regression because it hides the dynamic state. Using dynamic text within `aria-label` (e.g., `aria-label={`Filter, currently: ${state}`}`) successfully provides both purpose and current state to screen readers.\n**Action:** Always ensure `aria-label` includes dynamic state if the element visually displays that state.
