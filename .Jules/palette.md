## 2024-05-16 - Contextual ARIA labels for Icon Buttons
**Learning:** Screen reader users can easily get confused by multiple identical icon-only buttons in repeating elements (like tables or lists) if they do not have specific contextual data attached.
**Action:** When adding ARIA labels to icon-only buttons in repeating elements, I will inject contextual row data (e.g. description or ID) into the label so screen reader users can distinguish between them.
